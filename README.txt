Wikipedia Viewer

Built based on the Wikipedia challenge from freeCodeCamp
https://www.freecodecamp.com/challenges/build-a-wikipedia-viewer


Details:

Goal of the project was to utilize the wikipedia API
Enter a topic into the search and the page will load the 10 most relevant wikipedia articles
- If there are no revelant articles a popup will be displayed informing you no results were found

Technologies:
HTML, CSS/Bootstrap, JavaScript/jQuery
